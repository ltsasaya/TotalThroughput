# UI Layout

## Recommended Layout

### Top Bar
* Score
* Timer
* Current phase
* Dropped task count
* Warning state

### Left Panel
* Queue of waiting tasks
* Each task shows waiting time and size/type estimate

### Center Panel
* Cores shown as cards or lanes
* Idle or busy status
* Currently running task
* Progress bar

### Right Panel
* Live stats
* Throughput
* Queue length
* Utilization
* Average latency

### Bottom Panel or Modal
* Optional instructional hints
* Post-run explanation
* Graph summary
